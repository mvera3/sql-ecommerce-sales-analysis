Project: E-Commerce Sales Analysis

Tool Used:
SQL Server (SSMS)

Dataset:
Superstore Sales dataset containing order, product, customer,
sales, discount, and profit data.

Project Goal:
Analyze sales performance and identify profitability issues
using SQL queries.

Key Questions Analyzed:
1. Which product categories generate the most revenue?
2. Which customers generate the highest sales?
3. How do discounts affect profit?
4. Which products generate the biggest losses?

Key Insights:
• Technology category produced the highest sales.
• Large discounts (40–70%) often resulted in negative profit.
• The highest revenue customer generated a loss due to heavy discounts.
• Some high-value products caused significant losses when discounted.

Skills Demonstrated:
• SQL aggregation
• GROUP BY analysis
• Data filtering
• Business insight generation

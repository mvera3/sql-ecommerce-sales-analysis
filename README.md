# SQL E-Commerce Sales Analysis

## Project Overview
This project analyzes an e-commerce sales dataset using SQL to identify revenue trends, profitability issues, and top-performing customers.

## Tools Used
- SQL Server (SSMS)
- SQL

## Key Questions
- Which product categories generate the most revenue?
- Which customers generate the highest sales?
- How do discounts impact profitability?
- Which products generate the biggest losses?

## Key Insights
- Technology category generated the highest total revenue.
- Discounts above 40% often resulted in negative profit.
- One high-revenue customer generated a net loss due to heavy discounts.
- Some high-value products produced losses when sold with large discounts.

## Files in this Repository
- `sales_analysis_queries.sql` → SQL queries used for analysis
- `Screenshot.png` → example output from SQL analysis

## Dataset
The dataset used in this project is the Superstore Sales dataset, which contains order-level information including products, customers, discounts, sales, and profit metrics.
